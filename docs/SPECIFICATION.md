# 體育器材租借系統 - 規格說明文檔

## 📋 系統規格概述

### 項目名稱
體育器材租借系統 (Sports Equipment Rental System)

### 版本
v1.0

### 開發語言
C++17

### 開發目標
創建一個完整的體育器材租借管理系統，展示面向物件程設的核心概念。

## 🎯 功能需求

### 1. 器材管理模塊

#### 1.1 器材信息
系統支持三種器材類型：

**球類器材 (Ball)**
- 屬性：
  - 器材ID（自動生成）
  - 名稱
  - 球類型（籃球、足球、網球等）
  - 材料（橡膠、皮革等）
  - 重量（克）
  - 租借價格（$／天）
  - 可用狀態

**拍類器材 (Racket)**
- 屬性：
  - 器材ID（自動生成）
  - 名稱
  - 拍類型（網球拍、羽毛球拍等）
  - 拍面大小（L/M/S）
  - 弦張力
  - 租借價格（$／天）
  - 可用狀態

**保護裝備 (ProtectiveGear)**
- 屬性：
  - 器材ID（自動生成）
  - 名稱
  - 裝備類型（頭盔、護膝等）
  - 尺寸（S/M/L/XL）
  - 顏色
  - 租借價格（$／天）
  - 可用狀態

#### 1.2 器材操作
- **添加器材**：添加新的器材到系統
- **查看器材**：
  - 查看所有器材
  - 查看可租借器材
  - 查看詳細信息
- **編輯器材**：修改器材的信息（名稱、價格、描述）
- **刪除器材**：從系統中移除器材

### 2. 租借管理模塊

#### 2.1 租借紀錄
- 紀錄ID（自動生成）
- 器材ID
- 器材名稱
- 租借人名稱
- 租借日期
- 應歸還日期（基於租借天數自動計算）
- 租借天數
- 總費用（價格 × 天數）
- 歸還狀態

#### 2.2 租借操作
- **租借器材**：
  - 選擇器材
  - 輸入租借人名稱
  - 輸入租借天數
  - 自動計算費用
  - 生成租借紀錄
  - 更改器材狀態為"已租"

- **歸還器材**：
  - 輸入租借紀錄ID
  - 更新歸還日期
  - 改變器材狀態為"可租"
  - 保存交易完成記錄

- **查看租借紀錄**：
  - 查看所有租借紀錄
  - 按租借人查看紀錄
  - 查看待處理租借

### 3. 搜索和查詢模塊

#### 3.1 搜索功能
- **按名稱搜索**：查找包含特定名稱的器材
- **按分類搜索**：按器材類型（球類、拍類、保護裝備）搜索

#### 3.2 統計功能
- 系統中的總器材數
- 可租借器材數量
- 已租出器材數量
- 總租借紀錄數
- 已完成交易數
- 進行中的租借數
- 系統總收入

### 4. 數據持久化模塊

#### 4.1 文件操作
- **保存數據**：將器材和租借紀錄保存到 CSV 文件
- **加載數據**：系統啟動時自動加載已有數據
- **自動保存**：每次操作後自動保存數據

#### 4.2 文件格式

**equipment.csv**
```
ID,名稱,分類,價格,狀態,類型,詳細信息
1,籃球,球類,50.0,可租,球類-籃球|材料:橡膠|重量:600g|$50
```

**rental_records.csv**
```
紀錄ID,器材ID,器材名稱,租借人,租借日期,歸還日期,天數,費用,已歸還
1,1,籃球,張三,2024-01-01,2024-01-03,2,100.0,1
```

## 🏗️ 非功能需求

### 性能要求
- 系統應支持至少 1000 件器材
- 系統應支持至少 10000 條租借紀錄
- 操作響應時間 < 1 秒

### 可靠性要求
- 系統應正確處理異常情況
- 數據應持久化存儲
- 系統應在崩潰後恢復

### 可用性要求
- 提供清晰的菜單驅動界面
- 提供直觀的操作提示
- 支持跨平臺（Windows/Linux）

### 維護性要求
- 代碼結構清晰，便於維護
- 使用設計模式和最佳實踐
- 代碼有註釋和文檔

## 🔧 技術規範

### 編程語言
- C++17 標準

### 主要組件

#### 1. 類別設計

**Equipment（基類）**
```cpp
- 屬性: id, name, category, rentalPrice, isAvailable, description
- 方法: displayInfo() [虛擬], getDetails() [虛擬], setters, getters
```

**Ball（衍生自 Equipment）**
```cpp
- 額外屬性: ballType, material, weight
- 覆寫方法: displayInfo(), getDetails()
```

**Racket（衍生自 Equipment）**
```cpp
- 額外屬性: racketType, frameSize, stringTension
- 覆寫方法: displayInfo(), getDetails()
```

**ProtectiveGear（衍生自 Equipment）**
```cpp
- 額外屬性: gearType, size, color
- 覆寫方法: displayInfo(), getDetails()
```

**RentalRecord**
```cpp
- 屬性: recordId, equipmentId, equipmentName, renterName, rentalDate, 
        returnDate, rentalDays, totalCost, isReturned
- 方法: getters, setters, displayRecord(), toFileFormat()
```

**RentalSystem**
```cpp
- 屬性: equipment[], rentalRecords[], dataDirectory, nextEquipmentId, nextRecordId
- 方法: 器材管理、租借管理、文件操作、統計、搜索
```
