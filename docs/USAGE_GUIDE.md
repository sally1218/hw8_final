# 體育器材租借系統 - 使用指南

## 📖 快速開始

### 系統要求
- **操作系統**：Windows、Linux 或 macOS
- **編譯器**：GCC 7.0+ 或 MSVC
- **C++ 標準**：C++17

### 安裝和編譯

#### 步驟 1：准備環境
確保您已安裝：
- C++ 編譯器（g++、clang 或 MSVC）
- 文本編輯器或 IDE（Visual Studio Code、Visual Studio 等）

#### 步驟 2：編譯項目

**Windows 使用者**
```bash
# 切換至專案根目錄
cd d:\sally_school\大三\物件導向\hw8

# 執行自動偵測編譯批次檔 (會自動判斷 MinGW g++ 或 Visual Studio MSVC 並編譯)
build.bat
```

**Linux/macOS 使用者**
```bash
cd ~/path/to/hw8
make          # 或
cmake -B build && cmake --build build
```

#### 步驟 3：運行程序

**Windows 推薦執行方式 (避免中文亂碼)：**
雙擊專案目錄下的 **`START.bat`** 檔，或者在終端機中輸入以下指令啟動：
```bash
.\START.bat
```
*(這會自動將終端機編碼設定為 UTF-8 `chcp 65001`，確保所有的中文選單和圖案都完美顯示。)*

**如果是在 VS Code 的 Terminal (PowerShell) 中直接執行：**
若遇到中文亂碼，請在執行前輸入以下指令設定編碼：
```powershell
[Console]::OutputEncoding = [System.Text.Encoding]::UTF8; .\rental_system.exe
```

**Linux/macOS 執行方式：**
```bash
./rental_system
```

## 📋 功能使用說明

### 1️⃣ 查看所有器材

**操作步驟**
1. 在主菜單中選擇 `1`
2. 系統將顯示所有器材的列表

**顯示信息**
- 器材 ID
- 器材名稱
- 器材分類
- 租借價格（$/天）
- 租借狀態（可租借/已租出）
- 器材詳細信息

**示例輸出**
```
[1] ID:1|名稱:籃球|分類:球類|價格:50|狀態:可租
[2] ID:2|名稱:足球|分類:球類|價格:45|狀態:可租
[3] ID:4|名稱:網球拍|分類:拍類|價格:120|狀態:已租
```

### 2️⃣ 查看可租借器材

**操作步驟**
1. 在主菜單中選擇 `2`
2. 系統只顯示狀態為"可租借"的器材

**什麼時候使用**
- 快速查看有哪些器材可以租借
- 幫助客戶選擇器材

### 3️⃣ 器材管理

**操作步驟**
1. 在主菜單中選擇 `3`
2. 進入器材管理菜單

**器材管理菜單選項**
```
┌───────────── 器材管理菜單 ────────────┐
│                                      │
│  1. 添加新器材                       │

│  4. 返回主菜單                       │

```

#### 3.1 添加新器材

**操作步驟**
1. 選擇 `1. 添加新器材`
2. 選擇器材類型：
   - `1` - 球類器材
   - `2` - 拍類器材
   - `3` - 保護裝備
3. 輸入器材信息

**球類器材 (示例)**
```
器材名稱: 籃球
租借價格($/天): 50
球類型(如：籃球、足球): 籃球
材料(如：橡膠、皮革): 橡膠
重量(克): 600

✓ 器材已添加 (ID: 10)
```

**拍類器材 (示例)**
```
器材名稱: 乒乓球拍
租借價格($/天): 35
拍類型(如：網球拍): 乒乓球拍
拍面大小(L/M/S): M
弦張力(如：60磅): 高張力

✓ 器材已添加 (ID: 11)
```

**保護裝備 (示例)**
```
器材名稱: 滑冰護具套裝
租借價格($/天): 40
裝備類型(如：頭盔、護膝): 護具
尺寸(S/M/L/XL): L
顏色: 黑色

✓ 器材已添加 (ID: 12)
```

#### 3.2 編輯器材

**操作步驟**
1. 選擇 `2. 編輯器材`
2. 系統顯示所有器材
3. 輸入要編輯的器材 ID
4. 選擇要修改的項目：
   - `1` - 修改名稱
   - `2` - 修改租借價格
   - `3` - 修改描述
5. 輸入新值

**示例**

```
狀態: ✓ 可租借

選擇要修改的項目:
1. 名稱
2. 租借價格
3. 描述

請輸入: 2
新價格: 55
✓ 器材已更新
```

#### 4.1 租借器材

**操作步驟**
1. 選擇 `1. 租借器材`
2. 系統顯示所有可租借的器材
3. 輸入器材 ID
4. 輸入租借人名稱
5. 輸入租借天數
6. 系統自動計算費用

**完整示例**
```
╔════════════════════════════════════════╗
║              租借器材                  ║
╚════════════════════════════════════════╝

可租借的器材:
[1] 球類-籃球|材料:橡膠|重量:600g|$50
[2] 球類-足球|材料:皮革|重量:430g|$45

輸入器材ID: 1
租借人名稱: 張三
租借天數: 3

✓ 租借成功！
租借紀錄ID: 1
租借人: 張三
租借日期: 2024-01-15
應歸還日期: 2024-01-18
總費用: $150

按任意鍵繼續...
```

**費用計算**
```
總費用 = 租借價格 × 租借天數
例如: $50/天 × 3天 = $150
```

## 📝 常見任務說明

### 任務 1：完成一個租借交易

**步驟**
1. 主菜單 → 選擇 `2. 查看可租借器材`

## 🔧 故障排除

### 問題 1：程序無法啟動
**原因**：可能缺少必要的文件或編譯出錯

**解決方案**
1. 重新編譯程序
2. 確保所有頭文件都在 `include/` 目錄中
3. 檢查編譯器版本是否支持 C++17

### 問題 2：無法保存數據
**原因**：`data/` 目錄不存在或沒有寫入權限

**解決方案**
1. 確保 `data/` 目錄存在
2. 檢查目錄權限
3. 手動創建 `data/` 目錄

### 問題 3：租借時無法找到器材
**原因**：器材 ID 錯誤或該器材已被租借

**解決方案**
1. 選擇"查看可租借器材"確認可用的器材
2. 確認輸入的 ID 正確

### 問題 4：清屏功能不工作
**原因**：操作系統不同

**解決方案**
- 系統會根據操作系統自動選擇（Windows 用 `cls`，Linux 用 `clear`）
- 如果仍不工作，這不影響功能，只是顯示效果
